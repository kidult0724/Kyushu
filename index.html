景點資料整理自公開旅遊資訊，僅供行前參考，實際開放時間、交通與設施請以官方公告為準。上方「景點／餐廳／★我的收藏／全部」可切換檢視範圍，收藏的項目資料獨立保存，方便之後快速找回。新增資料時請分別使用「+ 新增景點」「+ 新增餐廳」「+ 新增住宿」按鈕，三者欄位不同（餐廳可填鄰近景點與平均消費、住宿可填類型與參考房價，皆可填地址或Google Maps連結讓地圖定位更準確）；餐廳卡片左側為橘色色條、住宿卡片為藍灰色條，與景點卡片（綠色色條）做區隔。標示「住宿」類型的項目為福岡、佐賀、長崎、熊本、大分五縣精選的真實飯店與旅館，房價僅為概略區間，實際房價會依房型、季節、訂房平台大幅波動，訂房前請務必至官方網站或訂房平台確認即時房價；住宿類項目不套用人潮估算與營業時間邏輯（飯店本身為全天營運）。各景點下方標示「餐廳」類型的項目為各地精選店家，營業時間、公休日仍可能異動，正式資訊請以「餐廳地點」按鈕查詢或現場確認為準。每張景點/餐廳卡片標籤列上的「🕐營業時間」為依類型（如歷史文化多為9-17點、溫泉設施多為8-21點、自然景觀類戶外景點視為全天開放）或料理類型推估的概略時段，並非逐店查證過的正確時間；超出時段會顯示「🌙 休息中」並停止顯示人潮百分比與長條，整張卡片也會變得較灰暗以便一眼分辨。「人潮」標籤並非即時人流感測數據（目前沒有公開的景點級即時人流API），而是綜合以下因子推估的參考百分比：①時段曲線（景點在下午較高、午餐晚餐時段較低；餐廳則相反）②星期幾（週日+5、週六+4、週五+3、週四+1、週三+0、週二-1、週一-2）③該縣觀光客量評分（依訪日旅客占比等公開資料排序，福岡+6〜宮崎-6）④該縣人口評分（福岡+3〜佐賀-3）⑤即時天氣與氣溫：晴天+5、多雲+2、陰天+0、雨天-5〜-10（依雨勢）、下雪-8、氣溫過高或過低額外-5，戶外景點受天氣影響加倍，餐廳與室內型景點則在壞天氣時人潮部分回升⑥景點自身知名度（越知名的景點基數分數越高，讓同縣市同類型的不同景點彼此有差異）。數字會隨你打開頁面的當下時間變動，僅供概略參考，觀光客量、人口排序與知名度評分皆為概略推估、非精確統計數據。天氣資訊由 Open-Meteo 免費 API 即時查詢，僅供概略參考，出發前請再次確認正式預報。各縣代表圖片為即時向維基百科抓取的當地景點真實照片，若圖片一時未顯示，代表當次抓取失敗，重新整理即可。地圖與路線連結會開啟 Google 地圖（不需登入），路線會以你目前所在位置為起點。收藏與新增的資料會保存在「這台裝置、這個瀏覽器」裡，重開此檔案仍會保留；但每個人、每台裝置的資料各自獨立，不會互相同步。
